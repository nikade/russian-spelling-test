# Отчет по проделанной работе (итерация 01)

## 1. Реализован базовый продукт
- Собрано веб-приложение «Тест на правильность написания» на чистом `HTML/CSS/JS`.
- Подключен Material Web 3 через CDN.
- Реализован сценарий из 10 заданий с последовательным прохождением и кнопкой `Дальше`.

## 2. Данные и логика теста
- Подготовлен справочник `data/tasks.json` на 30 слов.
- Для каждого задания заданы:
  - слово с пропуском (`..`)
  - правильная буква
  - варианты букв (2–4)
  - тип орфограммы
  - пояснение при ошибке
- Реализован случайный выбор 10 заданий при старте теста.
- Добавлена проверка ответа с мгновенной обратной связью:
  - `Верно` при правильном ответе
  - `Ошибка` + пояснение + правильное слово при неверном

## 3. Интерфейс и UX-доработки
- Увеличен размер букв на кнопках вариантов.
- После выбора ответов варианты скрываются.
- В слове после ответа пропуск заменяется на выбранную букву.
- При ошибке выбранная буква в слове выделяется красным и перечеркивается диагональной чертой.
- Для правильного слова добавлена крупная визуализация и выделение орфограммы.
- Убраны визуальные «скачки» экрана между состояниями вопрос/ответ:
  - зафиксирована высота блока обратной связи
  - сохранено место под блок вариантов

## 4. Итоговый экран
- Реализован экран завершения теста:
  - количество верных ответов
  - количество ошибок
  - процент правильных
  - список слов с ошибками
- Добавлена кнопка `Пройти еще раз` с новой случайной выборкой.

## 5. Качество и проверка
- Добавлены автотесты: `tests/quiz.test.js`, `tests/state.test.js`.
- Все тесты проходят (`7/7`).
- Проверен синтаксис ключевых JS-модулей.

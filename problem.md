# Техническое задание

## Web-приложение «Тест на правильность написания»

---

## 1. Назначение приложения

Web-приложение предназначено для тренировки орфографических навыков учащихся **2 класса** по следующим темам:

* безударные гласные в корне слова
* парные согласные в корне слова
* правописание ЖИ–ШИ
* правописание ЧА–ЩА
* правописание ЧУ–ЩУ
* словарные слова (без проверки)

Приложение используется в учебных и тренировочных целях.

---

## 2. Формат задания

### 2.1. Общий вид задания

* Пользователю показывается **одно слово с пропуском одной буквы**.
* Пропуск обозначается символами `..`.
* Ниже отображаются **варианты букв** для выбора (от 2 до 4).

Пример:
`б..рег`
[ а ] [ о ] [ е ]

---

### 2.2. Ограничения

* В одном слове **только один пропуск**.
* В одном задании проверяется **только одно правило**.
* Варианты ответов содержат **только буквы**, без слов.

---

## 3. Логика проверки ответа

### 3.1. Проверка

1. Пользователь выбирает одну букву.
2. Система сравнивает выбранную букву с правильной.
3. Результат проверки отображается **сразу после выбора**.

---

### 3.2. Верный ответ

При верном ответе:

* выводится сообщение: «Верно»
* автоматически становится доступна кнопка **«Дальше»**

Дополнительные пояснения **не показываются**.

---

### 3.3. Неверный ответ

При неверном ответе:

* выводится сообщение: «Ошибка»
* показывается **пояснение** (см. п. 4)
* показывается правильный вариант слова
* становится доступна кнопка **«Дальше»**

Повторный выбор в этом задании не предусмотрен.

---

## 4. Пояснение при ошибке

При неверном ответе система показывает **одно краткое пояснение**.
Формат пояснения зависит от типа орфограммы.

### 4.1. Безударная гласная в корне

Показывается проверочное слово.

Пример:
`Проверка: берег — берега`

---

### 4.2. Парная согласная в корне

Показывается проверочное слово, где согласный слышен отчётливо.

Пример:
`Проверка: гриб — грибы`

---

### 4.3. ЖИ–ШИ

Показывается правило в одной строке.

Пример:
`ЖИ–ШИ пишем с буквой И`

---

### 4.4. ЧА–ЩА

Пример пояснения:
`ЧА–ЩА пишем с буквой А`

---

### 4.5. ЧУ–ЩУ

Пример пояснения:
`ЧУ–ЩУ пишем с буквой У`

---

### 4.6. Словарные слова

Показывается пометка без объяснений.

Пример:
`Словарное слово. Нужно запомнить.`

---

## 5. Структура одного теста

* Один тест состоит из **10 заданий**.
* Задания показываются **последовательно**, по одному.
* Пропуск заданий невозможен.

Рекомендуемое соотношение:

* 6–7 заданий — безударные гласные / парные согласные
* 3–4 задания — ЖИ–ШИ, ЧА–ЩА, ЧУ–ЩУ
* не более 1–2 словарных слов на тест

---

## 6. Пользовательский интерфейс (поведение)

* Один экран = одно задание
* На экране:

  * слово с пропуском
  * варианты букв
  * область результата (верно / ошибка / пояснение)
  * кнопка «Дальше»
* Переход к следующему заданию возможен только после выбора буквы.

Дизайн и анимации в рамках данного ТЗ не регламентируются.

---

## 7. Требования к контенту

* Все слова соответствуют программе **2 класса**.
* Используются общеупотребительные и знакомые детям слова.
* Для каждого слова заранее задано:

  * правильное написание
  * тип орфограммы
  * текст пояснения при ошибке
